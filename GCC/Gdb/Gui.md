# GDB图形化调试

## 启动
1. tui enable  自动启动src窗口
2. layout src  自动启动src窗口
3. layout reg  启动寄存器窗口
4. layout asm  启动汇编代码
5. layout split  启动src和汇编窗口

## 关闭
1. tui disable
