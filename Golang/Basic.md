## Golang安装

1. 下载Golang
2. 管理员执行 rm -rf /usr/local/go && tar -C /usr/local -xzf [go1.17.7.linux-amd64.tar.gz]
3. 配置环境变量 export PATH=$PATH:/usr/local/go/bin
4. go version
