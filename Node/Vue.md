## Linux Ubuntu安装Vue
1. 安装node 
   - apt install nodejs
2. 安装npm、cnpm
   - apt install npm
   - sudo npm install -g cnpm --registry=https://registry.npm.taobao.org

3. 安装vue
   - sudo cnpm install -g vue-cli


## 创建Vue项目
- vue init webpack [项目名]
- cnpm install
- npm run dev
