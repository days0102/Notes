# Oracle简单函数
## 单行函数
### 字符函数
- ASCII(x)  
  - return 字符x的ASCII码值

- LENGTH(x)
  - return x的字符个数
- CONCAT(x,y)
  - 将y加到x上，并返回结果

- LOWER(x)
  - 把x中的字母转换成小写，并返回结果

- SUBSTR(x,start[,length])
  - 返回x中的一个字符串
  - 字符串长度从start开始
  - 可以指定长度
